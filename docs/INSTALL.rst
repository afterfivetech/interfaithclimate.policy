interfaithclimate.policy Installation
-------------------------------------

To install interfaithclimate.policy using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``interfaithclimate.policy`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        interfaithclimate.policy

* Re-run buildout, e.g. with:

    $ ./bin/buildout
